Polyglow は、エッセイ、ノート、リサーチの公開に向けた静的な多言語パブリッシングサイトです。安定した URL、読みやすい表示面、十分なメタデータ、CMS やサーバー runtime に依存しない長期運用可能なビジュアルシステムを重視しています。

プロジェクトは Astro を主要なレンダリング層として使い、必要なインタラクションは Astro コンポーネント内の小さな inline script で扱います。コンテンツは Markdown と MDX で管理されるため、fork、レビュー、バージョン管理、デプロイが容易です。
